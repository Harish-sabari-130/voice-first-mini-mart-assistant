# Voice-First Smart Inventory & Billing Assistant

A Tamil-first mobile application built using Flutter and Firebase for mini mart stores in Tamil Nadu.

## Features

- Image-based billing
- Smart optional invoice logic
- Voice assistant (Push-to-Talk)
- Inventory management
- Wholesale & Self-made product types
- Profit analysis
- Offline-first architecture
- Bilingual Tamil & English support

## Tech Stack

- Flutter
- Firebase Auth
- Firestore
- Hive (local storage)
- Speech-to-Text
- OCR integration

## Setup Instructions

1. Install Flutter SDK
2. Setup Firebase project
3. Add google-services.json
4. Run `flutter pub get`
5. Run `flutter run`
